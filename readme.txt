The Widget Import And Export plugin Exporting the Widget Data In JSON format than Import the JSON file on the import tab(Choose file).

== Description ==

**The easiest way to exporting widget setting in JSON format than Importing the file!**

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Once activated, youâ€™ll find widget import & export in the Dashboard Tools.

== How does this work? ==

The Widget Import and Export can be installed and activated like any other WordPress plugin.

Once installed, you can move on to **Tools -> Widget Import & Export** option.

== Manage The Settings ==

First of all, adding the widget in a sidebar, header and footer then go to the plugin setting **Tools ->Widget Import & Export** option

- There is show the all active widget there is checkbox widget then choose which widget you want then click on Export button. 

Then Generate the JSON file.

- Then go to Importing tab if you are not select(Choose file) the import button not visible.

once you are select JSON file on Import tab than Import button is visible.

- once you are Import the widgets(JSON file) there is show message which widget Imported or widget already Imported.
 

== Frequently Asked Questions ==

= How to get started with the Widget Import and Export? =
Install the Widget Import and Export plugin like any other WordPress plugin. Open Tools -> Widget Import and Export
Select the checkbox which widget you want to export.

= How I can Export the widget? =
You  add a widget into sidebar, Header or Footer then goto Tools -> Widget Import & Export
then select the widget and export the JSON file format.

= How I can Import Widget? =
Once you can be exporting JSON file.this JSON file Import tab(Choose file).
